# Enterprise_network
in this folder we have all configure for network in GNS3
الملخص
يتناول هذا المشروع تصميم وتنفيذ وتأمين شبكة مؤسسية متقدمة باستخدام برنامج المحاكاة GNS3 مع دعم VMware لتشغيل الأجهزة الافتراضية مثل pfSense و LibreNMS. تم تقسيم الشبكة إلى أقسام متعددة (IT، Finance، HR، Management، Guest) باستخدام شبكات افتراضية مستقلة  (VLANs)  لتحقيق عزل كامل لحركة المرور بين الأقسام وتعزيز الأمان على الطبقة الثانية والثالثة.
تم تطبيق توجيه ديناميكي (OSPF) مع Inter-VLAN Routing عبر واجهات  SVI، وتوزيع عناوين IP تلقائياً عبر مخدم DHCP مع حماية ضد هجمات DHCP Rogue باستخدام DHCP Snooping.  كما تم تعزيز أمان الطبقة الثانية بتفعيل Port Security، Dynamic ARP Inspection (DAI)، و Spanning Tree Protocol (STP) لمنع الحلقات.
أُضيف جدار حماية متقدم (pfSense) مع تكوين NAT و Firewall Rules، ونظام كشف اقتحام (IDS) باستخدام Snort ، بالإضافة إلى نظام مراقبة مركزي (LibreNMS) لجمع السجلات عبر Syslog وإصدار تنبيهات فورية. تم أيضاً تكوين خدمات مركزية (DNS، Web Server، FTP، Mail Server,ِِAAA) مع قوائم تحكم الوصول (ACLs) لتقييد حركة المرور بين الأقسام، ودعم VPN للوصول الآمن عن بُعد.
يهدف المشروع إلى تقديم حل شامل واقتصادي مفتوح المصدر يتوافق مع أفضل الممارسات العالمية Cisco مع التركيز على السياق المحلي في سوريا حيث ارتفعت الهجمات السيبرانية بشكل ملحوظ، مما يساهم في تقليل المخاطر ورفع كفاءة الشبكة.
